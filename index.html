MNIST Digit Recognition

MNIST Digit Recognition adalah sebuah program yang menggunakan model neural network untuk mengenali digit dari gambar yang diunggah. Program ini dibangun menggunakan Python dengan menggunakan TensorFlow dan Keras.

Fitur utama program ini adalah kemampuannya untuk memprediksi digit dari gambar yang diunggah pengguna. Anda cukup mengunggah gambar digit (dalam format PNG atau JPG) menggunakan formulir di bawah ini, kemudian program akan memberikan prediksi digit tersebut.
